Accesibility Audit

BlackBoard's website:

After performing a web accessibility evaulation on blackbaords home page using WAVE, this is the summary of results.

The erros of the home page include: 1 Empty heading, 1 Empty button, and 1 Broken ARIA Menu.
Here empty refers to the heading containing no content or the button having no value text. The ARIA Error refers to the menu not containing the required menu items by WAVE's standards. The rest include contrast errors which i have described previously and the warnings mention less accesibility for keyboard users and layout tables warnings for confusing keyboard users.
